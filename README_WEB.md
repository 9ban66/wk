# Yatori Web 部署说明

## 功能概览
- 提供一个简洁的 Web 页面，输入平台、账号、密码和平台地址后提交任务。
- 后台自动创建任务并执行登录与一键刷课流程。
- 当前已接入海旗、英华和学习通三类平台，且支持自定义平台地址。
- 可直接在 Linux 服务器上通过 Docker 或原生 Go 进程部署。

## 运行方式

### 1. 直接运行
```bash
go run . -addr :8080
```

### 2. Docker 运行
```bash
docker compose up -d --build
```

访问地址：
```text
http://<服务器IP>:8080/
```

## 使用流程
1. 打开首页。
2. 选择平台（海旗/英华）。
3. 输入账号、密码和平台地址。
4. 点击提交。
5. 后台即开始执行登录和一键刷课任务。

## 说明
- 这是一个可用的最小 Web 版骨架，已经把现有登录/刷课能力接到了后台任务流程中。
- 如果你后续想继续完善，可以把任务状态、日志、课程列表、停止任务等能力继续扩展到页面里。
